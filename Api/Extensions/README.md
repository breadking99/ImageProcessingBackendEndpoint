# Extensions

## Purpose
Contains extension methods that keep controllers clean.

## Content
- `EncodingTypeExtension`: maps `EEncodingType` to MIME content types.

## Magyar
### Cél
Olyan extension metódusok, amelyek tisztán tartják a controller kódot.

### Tartalom
- `EncodingTypeExtension`: az `EEncodingType` értékeket MIME content type-ra képezi.
