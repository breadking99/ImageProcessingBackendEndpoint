# Sample Images

Place your sample images in this folder. Examples:
- `emoji1.png`
- `emoji1.jpg`
- `emoji1-blur.png`

These files are copied to the test output directory by `Tests.csproj`.

## Magyar
Helyezd ide a minta képeket. Példák:
- `emoji1.png`
- `emoji1.jpg`
- `emoji1-blur.png`

A `Tests.csproj` ezeket a fájlokat átmásolja a teszt output mappába.
